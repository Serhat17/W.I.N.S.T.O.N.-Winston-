Privacy & Security

Your data stays on your machine. When using cloud providers, a privacy guard protects your personal information.

🔒 Local-First Architecture

All core functionality runs on your Mac via Ollama. No internet connection required for basic operation.

  • Local LLM inference (Ollama)
  • Local speech recognition (Whisper)
  • Local memory storage (ChromaDB)
  • No telemetry or analytics

🛡 Privacy Guard

When you choose to use cloud LLMs (OpenAI, Anthropic, Gemini), the privacy guard kicks in automatically.

  • Strips personal data from system prompts
  • Redacts passwords, API keys, credit cards
  • Sanitizes messages before sending to cloud
  • Warns when cloud provider is active

🛡 Safety System

Risk-based action confirmation prevents accidental damage.

  • SAFE — auto-approved (time, search, notes)
  • LOW — logged (system info, screenshots)
  • MEDIUM — confirmation required (open app, files)
  • HIGH — always confirms (email, commands)
  • BLOCKED — never allowed (destructive ops)

Get Started

Up and running in under 5 minutes on any Mac.

Requirements

💻

macOS

macOS 12+ (Monterey or later)

🤖

Ollama

Download from ollama.ai

🐧

Python 3.9+

Auto-installed by the setup script

🍺

Homebrew

Auto-installed if missing

1

Install Ollama & pull a model

Download Ollama from ollama.ai, then pull the default model:

ollama pull qwen2.5:7b
2

Install W.I.N.S.T.O.N.

Run the one-liner install script. It clones the repo, installs dependencies, and sets up everything:

curl -sL https://raw.githubusercontent.com/serhatbilge/W.I.N.S.T.O.N/main/install.sh | bash

Or manually:

git clone https://github.com/serhatbilge/W.I.N.S.T.O.N.git ~/W.I.N.S.T.O.N
cd ~/W.I.N.S.T.O.N && bash setup_mac.sh
3

Configure (optional)

Edit config/settings.yaml to enable channels, set API keys, or customize behavior. The defaults work out of the box.

4

Run

Choose your preferred mode:

# Text mode (CLI)
python -m winston.main --mode text

# Voice mode
python -m winston.main --mode voice

# Server mode (Web UI + all channels)
python -m winston.main --mode server
