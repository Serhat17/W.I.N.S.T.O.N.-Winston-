Add Your Own Skill in 30 Lines

winston/skills/my_skill.py 
from winston.skills.base import BaseSkill, SkillResult

class MySkill(BaseSkill):
    name = "my_skill"
    description = "What it does — the LLM reads this"
    parameters = {
        "query": {"type": "string", "description": "Input"}
    }

    def execute(self, query="", **kwargs):
        return SkillResult(success=True, message=f"Done: {query}")

Register it, and the LLM automatically routes matching requests to your skill. No configuration needed.

Get Started in 2 Minutes

Install Winston

1

Install Winston via Homebrew

brew install serhatbilge/tap/winston
2

Install a local AI model

brew install ollama && ollama serve &
ollama pull qwen2.5:7b
3

Launch the setup wizard

winston --setup

The wizard guides you through connecting Telegram, adding API keys, and choosing your default mode.

4

Done. Just type:

winston
1

Run the one-line installer

curl -fsSL https://raw.githubusercontent.com/serhatbilge/W.I.N.S.T.O.N..S/main/install.sh | bash

This automatically installs Python, Ollama, a default AI model, and Winston. An interactive setup wizard guides you through connecting Telegram, adding API keys, etc.

2

Done. Just type:

winston

What the installer does:

  1. Checks internet connectivity & Xcode CLI Tools
  2. Installs Homebrew (if missing on macOS)
  3. Installs Python 3.12, Ollama, PortAudio, FFmpeg
  4. Clones the repo, creates a Python virtualenv, installs all dependencies
  5. Pulls a default AI model (qwen2.5:7b)
  6. Runs the interactive setup wizard
1

Prerequisites

# macOS
brew install python@3.12 ollama portaudio ffmpeg
ollama serve &
ollama pull qwen2.5:7b
2

Clone & install

git clone https://github.com/serhatbilge/W.I.N.S.T.O.N..S.git
cd W.I.N.S.T.O.N..S
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
3

Configure & run

cp .env.example .env   # Add API keys (all optional)
winston --setup        # Interactive setup wizard
winston                # Start chatting

Frequently Asked Questions

Any Mac with Apple Silicon (M1/M2/M3/M4) runs local models comfortably. 8GB RAM handles 7B models; 16GB+ handles 14B+ models. On Linux, a decent GPU helps but CPU-only works too. If your hardware is limited, just connect a cloud provider — the Winston skills still run locally.

Winston uses Playwright to control a real Chromium browser. The AI sees a structured snapshot of the page (not screenshots), decides which element to click/type/scroll, and executes actions. Winston is designed for macOS and Linux. It should work on Windows with WSL2, but it's not officially tested. Some macOS-specific features (Apple Calendar, system_control) won't work on other platforms.
