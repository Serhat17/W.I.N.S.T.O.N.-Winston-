Download W.I.N.S.T.O.N.

Your own AI assistant — like Jarvis, but open-source.
Runs locally with full privacy.

🍺 Homebrew (macOS) Recommended

brew install serhatbilge/tap/winston

Then run: winston setup

⚡ One-Line Install (macOS & Linux)

curl -fsSL https://raw.githubusercontent.com/serhatbilge/W.I.N.S.T.O.N..S/main/install.sh | bash

Installs Python, Ollama, a default AI model, and Winston. Interactive setup wizard included.

🐙 Manual (Git Clone)

git clone https://github.com/serhatbilge/W.I.N.S.T.O.N..S.git ~/Winston
cd ~/Winston && python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
winston setup

After Installing

# Start chatting winston # Start web UI + Telegram + Discord winston server # Voice mode (microphone + speaker) winston voice # Run setup wizard winston setup
